# Product Overview

MFER形式の心電図データを読み込み、心不全（MI: Myocardial Infarction）のリスクを推論する医療支援サービス。
医療従事者が心電図データから迅速かつ客観的に心不全リスクを評価できるよう支援する。

## Core Capabilities

1. **MFERデータ自動取り込み**
   指定フォルダを監視し、新規MFERファイルを検出→患者データ・検査データをDBへ自動登録

2. **心電図データ変換パイプライン**
   MFER形式 → CSV正規化 → Vision Transformer用画像生成

3. **AI心不全リスク推論**
   Vision Transformerモデルによる心電図画像からの心不全リスク判定

4. **患者・検査データ検索**
   登録済み患者情報のフィルタリング・一覧表示

5. **統合結果表示**
   患者情報・心電図・推論結果を一画面で確認

## Target Use Cases

| # | ユースケース |
|---|-------------|
| UC-1 | システムは指定フォルダを監視し、新規MFERデータ追加時に検査データ・患者データをDBへ自動登録する |
| UC-2 | システムはユーザー認可（パスワード入力）により利用許可を制御する |
| UC-3 | システムユーザーはDB登録済みの検査データを一覧表示できる |
| UC-3-extend | システムユーザーは患者情報で検査データにフィルター条件を指定して一覧する |
| UC-4 | システムユーザーは検査データを選択し、結果・患者データ・心電図を参照できる |
| UC-4-extend | システムはシステムユーザーの検査データ選択時に、心不全推論を実行する |

## Value Proposition

- **早期発見支援**: AIによる客観的なリスク評価で心不全の早期発見を支援
- **業務効率化**: MFERデータの自動取り込みにより手動登録作業を削減
- **判断支援**: 医療従事者の診断判断を補助する参考情報を提供（最終判断は医師が行う）
- **医療アクセス拡大**: 血液検査設備がない離島・僻地でも心電図のみで心不全リスク評価が可能

## 対象ユーザー（システムユーザー）

- 循環器内科医
- 心電図検査技師
- 一般的な内科医師
  - 健康診断等で非検査者に検査結果を説明する責務がある医師
- 医療機関の診療支援スタッフ
