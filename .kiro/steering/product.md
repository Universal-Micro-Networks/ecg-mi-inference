# Product Overview

MFER形式の心電図データを読み込み、心筋梗塞（MI: Myocardial Infarction）のリスクを推論する医療支援サービス。
医療従事者が心電図データから迅速かつ客観的に心筋梗塞リスクを評価できるよう支援する。

## Core Capabilities

1. **MFERデータ自動取り込み**
   指定フォルダを監視し、新規MFERファイルを検出→患者データ・診察データをDBへ自動登録

2. **心電図データ変換パイプライン**
   MFER形式 → CSV正規化 → Vision Transformer用画像生成

3. **AI心筋梗塞リスク推論**
   Vision Transformerモデルによる心電図画像からの心筋梗塞リスク判定

4. **患者・診察データ検索**
   登録済み患者情報のフィルタリング・一覧表示

5. **統合結果表示**
   患者情報・心電図・推論結果を一画面で確認

## Target Use Cases

| # | ユースケース |
|---|-------------|
| UC-1 | システムは指定フォルダを監視し、新規MFERデータ追加時に診察データ・患者データをDBへ自動登録する |
| UC-2 | システムはユーザー認証（パスワード入力）により利用許可を制御する |
| UC-3 | システムユーザーはDB登録済みの診察データを一覧表示できる |
| UC-4 | システムユーザーは患者情報で診察データをフィルタリングできる |
| UC-5 | システムユーザーは診察データを選択し、心筋梗塞推論を実行、結果・患者データ・心電図を参照できる |

## Value Proposition

## Value Proposition

- **早期発見支援**: AIによる客観的なリスク評価で心筋梗塞の早期発見を支援
- **業務効率化**: MFERデータの自動取り込みにより手動登録作業を削減
- **判断支援**: 医療従事者の診断判断を補助する参考情報を提供（最終判断は医師が行う）
- **医療アクセス拡大**: 血液検査設備がない離島・僻地でも心電図のみで心筋梗塞リスク評価が可能
